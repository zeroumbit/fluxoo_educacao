# ⚡ Guia Rápido de Início

## 🚀 Em 3 Passos

### 1️⃣ Instalar Dependências

```bash
npm install
```

### 2️⃣ Verificar Ambiente

```bash
npm run check-env
```

### 3️⃣ Rodar Servidor

```bash
npm run dev
```

🌐 **Acesse:** http://localhost:5173

---

## 🔐 Primeiro Acesso

### Configurar Super Admin

1. Acesse: https://app.supabase.com/project/phuyqtdpedfigbfsevte/auth/users
2. Clique em **Add user** → **Create new user**
3. Preencha:
   - **Email:** `zeroumti@gmail.com`
   - **Senha:** (defina uma senha)
4. Clique em **Create user**
5. Faça login em: http://localhost:5173/login
